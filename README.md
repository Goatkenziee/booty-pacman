# Booty Pacman 🍑

A cheeky Pac-Man game where the Pac-Man is a booty cheek! Built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.

## Controls

- Arrow keys to move
- Gobble dots and avoid ghosts
- Power pellets let you eat ghosts
